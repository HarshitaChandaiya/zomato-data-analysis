# 🍽️ Zomato Data Analysis

## 📌 Objective
Analyze restaurant data to understand customer preferences, pricing trends, and rating patterns.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)

## 📊 Key Analysis
- Top restaurant locations
- Online ordering distribution
- Cost vs rating analysis
- Popular cuisines

## 💡 Key Insights
- Mid-range restaurants have better ratings
- Online ordering is widely preferred
- No strong relationship between cost and rating
- Certain locations dominate restaurant density

## 📁 Dataset
Zomato Bangalore Restaurants dataset

## 🚀 Conclusion
This project provides insights into restaurant performance and customer behavior, useful for business decision-making.
